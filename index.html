The method

Three principles. Every villa, every time.

01
Rooted

The land shapes the house. The slope, the trees, the morning sun aren't constraints. They are the program.

02
Singular

One site, one vision, one time. We don't reuse plans, we don't run parallel projects. Each home is conceived as if it were the only one we will ever build.

03
Local

Stone from Andalusian quarries. Solid wood from trusted mills. Partners chosen within a short drive, never a long flight.

The studio

Sadala is a family studio in Málaga, second generation since 1990. We don't build for clients. We build the homes we'd want to live in ourselves, one at a time, then we sell them.
